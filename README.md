# 🛒 MERN E-Commerce Project

## 🚀 Features
- User Authentication (JWT)
- Product Listing
- Add to Cart
- Cart Management
- Stripe Payment Integration
- Contact Form

## 🛠 Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
- Database: MongoDB

## 📦 Installation

### Backend
cd Backend
npm install
npm start

### Frontend
cd frontend
npm install
npm run dev
